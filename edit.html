チェックポイント

  • 種類: "checkpoint" で固定
  • 名前: アルファベット大文字で表示名 (例："CHECKPOINT1")
  • プロパティ: id => 0から連番でチェックポイントの数,

敵

  • 種類: "enemy" で固定
  • 名前: 使用されない(なんでもいい)
  • プロパティ: 下記で説明。typeは必須

敵の種類ごとのプロパティ

※ p1,p2は、省略すると0が使用される

名前 type p1 p2 説明
スライム SLIME 方向( 1:右向き, -1:左向き )
ウィルオーウィスプ WOW 方向( 0:上, 1:右, 2:下, 3:左 ) 速度( 1/16[px/frame] )
エレベーター ELEVATOR 方向( 0:上, 1:右, 2:下, 3:左 ) 速度( 1/16[px/frame] )
動かせる岩 BLOCK
カエル FROG 方向( 1:右, 3:左 )
クッキー FROG 初期角度( 0〜255, 右から時計回り ) 回転速度( 正で右回転、負で左回転。単位=1/256[円周/frame]) 回るウィルオーウィスプ。"p3"で回転の大きさを指定(3=標準で小さい値ほど大きな円を描く)
