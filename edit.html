チェックポイント

  • 種類: "checkpoint" で固定
  • 名前: アルファベット大文字で表示名 (例："CHECKPOINT1")
  • プロパティ: id => 0から連番でチェックポイントの数,

敵

  • 種類: "enemy" で固定
  • 名前: 使用されない(なんでもいい)
  • プロパティ: 下記で説明。typeは必須

敵の種類ごとのプロパティ

※ p1,p2は、省略すると0が使用される

名前 type p1 p2 説明
スライム SLIME 方向( 1:右向き, -1:左向き )
ウィルオーウィスプ WOW 方向( 0:上, 1:右, 2:下, 3:左 ) 速度( 1/16[px/frame] ) 壁で反射する。
エレベーター ELEVATOR 方向( 0:上, 1:右, 2:下, 3:左 ) 速度( 1/16[px/frame] )
