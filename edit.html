チェックポイント

  • 種類: "checkpoint" で固定
  • 名前: アルファベット大文字で表示名 (例："CHECKPOINT1")
  • プロパティ: id => 0から連番でチェックポイントの数,

敵

  • 種類: "enemy" で固定
  • 名前: 下表のtypeのいずれかを指定する

敵の種類ごとのプロパティ

※ p1,p2は、省略すると0が使用される

※ 方向は 0:上, 1:右, 2:下, 3:左

※ 速度は単位が書かれていなければ 1/16[px/frame]

名前 type p1 p2 p3 説明
スライム SLIME 方向( 1:右向き, -1:左向き ) ※こいつだけ特殊 下矢印の場所で溶ける
ウィルオーウィスプ WOW 方向 速度 方向指定ブロックで曲がる
エレベーター ELEVATOR 方向 速度 幅( 0: 2キャラ、 1: 1キャラ)
カエル FROG 方向(右左) ジャンプの高さ(デフォルト:32)
クッキー COOKIE 初期角度( 0〜255, 右から時計回り ) 回転速度( 正で右回転、負で左回転。単位=1/256[円周/frame]) 回転の大きさ(3=標準で小さい値ほど大きな円を描く) 回るウィルオーウィスプ
ランプの火 LAMP
ガスさん GAS 方向 速度
テレサみたいなゴースト GHOST 速度
スイッチ SWITCH フラグ番号(0..31のいずれか)
フラグ開閉ドア FLAGED_DOOR フラグ番号(0..31のいずれか)
無限湧きポイント PORTAL (沸かせるモンスターのp1) (沸かせるモンスターのp2) 周期( 1[sec] ) typeに "portal:slime", "portal:wow" など沸かせるモンスターの種類をいれる
